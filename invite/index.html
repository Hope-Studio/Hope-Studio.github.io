                      # 优势

                      # 我们可以

                      • 如果大学 4 年全程跟下来，你将有能力拿到大厂的 offer。✅

                      • 工作室可为优秀者提供部分大厂的内部推荐。✅

                      • 部分开发任务提供一定薪酬。✅

                      # 我们不能

                      • 我们没有正式注册公司没有公章，无法开具正式的实习证明。⚠️

                      • 由于工作室具有盈利性质且室长已经毕业，我们未在校内注册社团。⚠️

                      • 我们不发工资，您只有承接项目才可能有薪酬。⚠️
